# Freshman_Web-Development-Assignment_Podcast-Website

大一网页设计作业：个人播客网站《听见成长》

## 项目简介

`听见成长` 是一个面向大学生和年轻学习者的静态个人播客网站，主题围绕学习方法、阅读分享、时间管理和成长感悟展开。

项目使用：

- `HTML5`
- `CSS3`
- `DIV + CSS` 布局
- 开发工具：`VS Code`

本项目当前已经完成静态页面开发，相关文档见：

- `docs/project-definition.md`
- `docs/wireframe.md`
- `docs/project-report.md`

## 页面规划

- `html/index.html`：首页
- `html/about.html`：个人简介页
- `html/episodes.html`：节目列表页
- `html/episode-detail.html`：单期节目详情页
- `html/resources.html`：成长资源页
- `html/contact.html`：留言表单页

## 参考原型

- 参考网站：`https://www.mypodops.com/`

本项目只借鉴其简洁播客首页结构，包括主视觉区、节目卡片区和清晰导航，不复刻复杂交互。
