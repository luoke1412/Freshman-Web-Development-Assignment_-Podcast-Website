# 听见成长个人网站开发项目说明书

## 1. 项目名称

听见成长

## 2. 项目类型

静态个人播客网站

## 3. 项目背景

本项目是网页设计课程作业，要求学生独立完成网站策划、分析、制作和调试的完整流程。为了满足作业中对文字、图片、音频、表单、多页面结构和页面美化的要求，我选择了个人播客网站作为主题。

## 4. 选题原因

选择播客网站有三个原因：

1. 播客网站适合展示文字、图片和音频内容。
2. 网站页面结构清晰，适合使用 `DIV + CSS` 进行布局。
3. 成长类主题积极向上，具有一定实用价值。

## 5. 网站主题

本网站主题为“成长型个人播客网站”，主要分享：

1. 学习方法
2. 阅读体会
3. 时间管理
4. 校园生活感悟

## 6. 目标用户

1. 大学生
2. 喜欢成长类内容的年轻人
3. 想通过轻松内容获得学习启发的浏览者

## 7. 页面设计

本网站共设计 6 个页面：

1. `index.html` 首页
2. `about.html` 关于我页面
3. `episodes.html` 节目列表页面
4. `episode-detail.html` 节目详情页面
5. `resources.html` 成长资源页面
6. `contact.html` 联系我页面

其中首页为重点页面，关于我页面用于展示个人简介，联系我页面用于完成表单设计要求。

## 8. 技术实现

本项目使用以下技术完成：

1. `HTML5`：负责页面结构
2. `CSS3`：负责页面样式和美化
3. `DIV + CSS`：完成页面布局

布局时综合使用了：

1. 标准文档流
2. 盒子模型
3. 定位
4. 过渡效果
5. 圆角和阴影

## 9. 页面风格

网站整体风格以清新、简洁、温暖为主，颜色选择偏柔和，主要使用紫色和浅色背景来表现成长类内容的轻松氛围。

## 10. 网站包含的网页元素

本项目中使用了以下网页元素：

1. 文字
2. 超链接
3. 图像占位区
4. 列表
5. 表单
6. 音频播放器预留区

后续可以继续加入真实图片和音频素材，使页面更完整。

## 11. 文件夹结构

```text
podcast-site/
├─ index.html
├─ about.html
├─ episodes.html
├─ episode-detail.html
├─ resources.html
├─ contact.html
├─ css/
├─ img/
├─ audio/
└─ docs/
```

这种结构便于管理 HTML、CSS、图片、音频和文档内容。

## 12. 开发流程

本项目开发流程如下：

1. 明确网站主题和目标用户
2. 设计页面结构和原型图
3. 建立项目文件夹结构
4. 先完成首页布局
5. 再补充其他页面
6. 统一样式风格
7. 最后进行调试和整理文档

## 13. 项目亮点

1. 页面数量满足课程要求
2. 首页内容完整，结构清晰
3. 包含个人简介页和表单页
4. 整体风格统一，适合展示成长类主题
5. 后续补充素材后可直接作为完整个人网站展示

## 14. 后续完善方向

1. 替换真实播客封面图
2. 加入个人音频素材
3. 补充更多节目详情内容
4. 优化移动端显示效果

## 15. 项目总结

通过本次作业，我完成了一个多页面静态网站的设计与实现过程，也进一步练习了 HTML 和 CSS 的综合运用。这个项目既满足了课程要求，也帮助我熟悉了从网站策划到页面实现的完整流程。
