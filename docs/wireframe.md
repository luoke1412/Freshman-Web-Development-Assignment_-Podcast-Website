# 听见成长低保真原型

## 1. 原型借鉴建议

这次原型建议借鉴简单播客首页结构，重点借鉴以下元素：

1. 顶部导航横向排布
2. 首屏左文右图布局
3. 中间使用 3 列卡片展示节目
4. 下方增加音频试听区和资源区

参考网址：`https://www.mypodops.com/`

你画原型时只需要借鉴结构，不需要照着做复杂内容。

## 8. 最容易落地的原型画法

你答辩前可以用以下任一方式快速完成原型：

1. 用纸和笔直接画方框图
2. 用 Word 插入矩形排版
3. 用 PPT 画页面区块

建议做法：

1. 重点画首页
2. 再补关于我页和联系我页
3. 其余页面保持同样结构即可

这套原型已经足够用于课程答辩。
